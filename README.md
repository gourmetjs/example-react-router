# Developing an Isomorphic App using React Router

This is a companion repo of this [article](www.example.com).

## Install

```sh
git clone https://github.com/gourmetjs/example-react-router
cd example-react-router
npm install
```

## Build

```sh
npm run build
```

## Start

```sh
npm start
```

## Watch

```sh
npm run watch
```
